# ESOE DB Backend

## 執行說明

請先安裝 node

### Installation

安裝套件

```bash
npm install
```

### Running the app

```bash
npm run start
```

Then open <http://localhost:3000/>

## Description （以下為開發使用）

本專案使用 [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
